# Category Context: Laptops

Subcategory of: `electronics`

## Key Buying Criteria

These are the questions and specs that most often determine whether a laptop is right for a buyer.
Always ask about or research these when helping someone choose a laptop.

### Critical questions to ask in intake
- **Primary use case**: General/school? Creative work (video, photo, design)? Gaming? Programming?
  Heavy multitasking? Business travel? Each shifts the recommendation dramatically.
- **Operating system preference**: Mac vs. Windows vs. ChromeOS vs. Linux. Ask if undecided —
  don't assume. Factor in existing ecosystem (iPhone/iPad → Mac affinity; existing Windows software
  library → Windows may be necessary).
- **Screen size preference**: 13–14" (portable) vs. 15–16" (balanced) vs. 17"+ (desktop replacement).
  Also ask: is weight a priority?
- **Performance tier needed**: Light (Office/web/school) → mid (programming, light creative) →
  heavy (video editing, 3D, gaming, ML). Ask what specific software they run.
- **RAM**: 8 GB is minimum today; 16 GB recommended for most; 32 GB+ for video/3D/ML workloads.
- **Storage**: 256 GB is cramped; 512 GB is comfortable; 1 TB+ for media creators.
- **GPU needs**: Discrete GPU is critical for gaming, 3D rendering, and some ML tasks. Integrated
  is fine for everything else.
- **Battery life expectations**: All-day use away from outlet? Or mostly plugged in?
- **Connectivity**: USB-A still needed? Thunderbolt? HDMI? SD card reader?

### Key specs to include in comparison table
- CPU (chip family: Apple M-series, Intel Core Ultra, AMD Ryzen)
- RAM and whether it's upgradeable
- Storage and whether it's upgradeable
- Screen: size, resolution, refresh rate, panel type (OLED > IPS > TN)
- GPU: integrated vs. discrete, VRAM if discrete
- Battery life (hrs, real-world not spec sheet)
- Weight
- Price (new)

### Common pitfalls
- Soldered RAM/SSD: many modern laptops (especially thin/light) have non-upgradeable components.
  Always flag this — it means the config at purchase is the config forever.
- "Gaming laptop" for design work: high-powered but heavy, poor color accuracy, short battery.
- Base storage (256 GB) fills up fast for students and creators.
- MacBook Air vs. MacBook Pro confusion: Air has no fan (throttles under sustained load),
  Pro has fan and more ports. For video editing, Pro is meaningfully better.
- Windows on ARM (Qualcomm Snapdragon X): impressive battery/performance but software
  compatibility gaps — always check if their key apps run natively.
- Chromebook limitations: only web apps + Android apps, no full desktop software.

### Price tiers (as of 2024–2025)
- Budget: $300–$600 (Chromebooks, entry Windows)
- Mid-range: $600–$1,000 (most students/general users live here)
- Performance: $1,000–$1,500 (MacBook Air M3/M4, Dell XPS, ThinkPad X1)
- Enthusiast/Pro: $1,500–$2,500+ (MacBook Pro, Razer Blade, creator laptops)
